# 一、认识 VB

## 1. VB 是什么

VB 是 Visual Basic 的简称，是一种可视化、事件驱动的程序设计语言。

“Visual” 表示可视化，开发者可以通过拖放控件来设计程序界面；“Basic” 表示它继承了 BASIC 语言简单易学的特点。

使用 VB 编程时，通常不是从空白代码开始，而是先设计一个窗体界面，再给窗体或控件编写事件代码。例如：

- 点击按钮时执行某段代码。
- 在文本框中输入内容后进行计算。
- 选择某个选项后显示不同结果。
- 定时器每隔一段时间触发一次事件。

## 2. VB 程序的基本特点

VB 程序有几个非常重要的特点：

### 2.1 可视化设计

传统程序往往先写代码，再运行看效果。VB 则可以先创建窗体，拖入按钮、文本框、标签等控件，再编写逻辑代码。

常见界面元素包括：

| 控件 | 英文名称 | 作用 |
| --- | --- | --- |
| 窗体 | Form | 程序窗口，是控件的容器 |
| 标签 | Label | 显示说明性文字 |
| 文本框 | TextBox | 输入或显示文本 |
| 命令按钮 | CommandButton | 用户点击后执行命令 |
| 单选按钮 | OptionButton | 多个选项中选择一个 |
| 复选框 | CheckBox | 可以同时选择多个选项 |
| 列表框 | ListBox | 显示列表项目 |
| 组合框 | ComboBox | 下拉选择或输入 |
| 定时器 | Timer | 按时间间隔触发事件 |

### 2.2 事件驱动

VB 程序不是总是从上到下执行所有代码，而是由用户操作或系统事件触发代码。

例如：

```vb
Private Sub Command1_Click()
    MsgBox "你点击了按钮"
End Sub
```

这段代码的意思是：当用户点击 `Command1` 按钮时，执行 `MsgBox` 语句，弹出提示框。

其中：

- `Command1` 是按钮对象名。
- `Click` 是点击事件。
- `Private Sub ... End Sub` 是一个事件过程。

### 2.3 面向对象的基本思想

VB 中的窗体、按钮、文本框等都是对象。对象通常有三个重要概念：

| 概念 | 含义 | 示例 |
| --- | --- | --- |
| 属性 | 对象的特征 | 按钮的 Caption、文本框的 Text |
| 方法 | 对象能执行的动作 | 窗体的 Show、列表框的 AddItem |
| 事件 | 对象发生的动作 | Click、Load、Change |

例如：

```vb
Text1.Text = "欢迎学习 VB"
Command1.Caption = "确定"
Form1.BackColor = vbWhite
```

这些语句都在修改对象的属性。

## 3. VB 程序的组成

一个完整的 VB 工程通常由以下部分组成：

| 组成部分 | 说明 |
| --- | --- |
| 工程 Project | 一个应用程序的整体 |
| 窗体 Form | 程序界面 |
| 控件 Control | 窗体上的按钮、文本框等 |
| 模块 Module | 保存通用代码 |
| 过程 Procedure | 一段可执行代码 |
| 事件过程 Event Procedure | 响应事件的代码 |

最常见的程序结构如下：

```vb
Private Sub Form_Load()
    '窗体加载时执行
End Sub

Private Sub Command1_Click()
    '按钮被点击时执行
End Sub
```

## 4. 第一个 VB 程序

目标：点击按钮，在标签中显示“Hello, VB!”。

### 4.1 界面设计

在窗体上放置：

| 控件 | Name 属性 | Caption 或 Text |
| --- | --- | --- |
| Label | Label1 | 空 |
| CommandButton | Command1 | 显示问候 |

### 4.2 编写代码

双击按钮，输入：

```vb
Private Sub Command1_Click()
    Label1.Caption = "Hello, VB!"
End Sub
```

### 4.3 程序分析

这段代码只在用户点击按钮时执行。

`Label1.Caption` 表示标签上显示的文字。把它设置为 `"Hello, VB!"` 后，窗体上的标签文字就会改变。

# 二、VB 程序设计基础

## 1. 工程、窗体与控件

### 1.1 工程

工程是一个程序项目。一个工程可以包含多个窗体、模块、类模块和资源文件。

对于初学者，最常见的是一个工程中只有一个窗体 `Form1`。

### 1.2 窗体

窗体是程序运行时看到的窗口。常用属性包括：

| 属性 | 说明 | 示例 |
| --- | --- | --- |
| Name | 窗体对象名 | Form1 |
| Caption | 标题栏文字 | 学生成绩计算 |
| Width | 宽度 | 6000 |
| Height | 高度 | 4000 |
| BackColor | 背景颜色 | vbWhite |
| StartUpPosition | 启动位置 | 2 表示屏幕中央 |

常用事件：

| 事件 | 触发时机 |
| --- | --- |
| Load | 窗体加载时 |
| Click | 点击窗体时 |
| Unload | 窗体关闭时 |

### 1.3 控件

控件是放在窗体上的界面元素。

控件也有属性、事件和方法。例如按钮：

```vb
Command1.Caption = "计算"
Command1.Enabled = True
Command1.Visible = True
```

其中：

- `Caption` 表示按钮显示文字。
- `Enabled` 表示按钮是否可用。
- `Visible` 表示按钮是否可见。

## 2. 命名规则与命名习惯

VB 中变量、控件、过程的名字通常要遵守以下规则：

1. 必须以字母开头。
2. 可以包含字母、数字和下划线。
3. 不能使用 VB 关键字。
4. 名字中不要出现空格。
5. 尽量做到见名知义。

不推荐：

```vb
Dim a As Integer
Dim b As Integer
```

更推荐：

```vb
Dim studentAge As Integer
Dim totalScore As Single
```

经典 VB 中常见控件命名前缀：

| 前缀 | 控件 | 示例 |
| --- | --- | --- |
| frm | Form | frmMain |
| lbl | Label | lblResult |
| txt | TextBox | txtName |
| cmd | CommandButton | cmdCalculate |
| opt | OptionButton | optMale |
| chk | CheckBox | chkMusic |
| lst | ListBox | lstStudents |
| cbo | ComboBox | cboClass |
| tmr | Timer | tmrClock |

如果教材或考试环境默认控件名为 `Text1`、`Command1`，也可以继续使用默认命名。

## 3. 数据类型

变量需要保存数据，而不同数据适合用不同类型。

### 3.1 常见数据类型

| 数据类型 | 说明 | 示例 |
| --- | --- | --- |
| Integer | 整数 | 18 |
| Long | 长整数 | 100000 |
| Single | 单精度小数 | 3.14 |
| Double | 双精度小数 | 3.1415926 |
| String | 字符串 | "Tom" |
| Boolean | 逻辑值 | True、False |
| Date | 日期时间 | #2026-04-29# |
| Variant | 可变类型 | 可以保存多种数据 |

示例：

```vb
Dim age As Integer
Dim score As Single
Dim studentName As String
Dim isPass As Boolean

age = 18
score = 92.5
studentName = "李华"
isPass = True
```

### 3.2 字符串

字符串必须用英文双引号括起来：

```vb
Dim msg As String
msg = "欢迎学习 VB"
```

字符串可以连接：

```vb
Dim name As String
name = "小明"
MsgBox "你好，" & name
```

VB 中字符串连接推荐使用 `&`。

### 3.3 逻辑值

逻辑值只有两个：

```vb
True
False
```

常用于条件判断：

```vb
Dim isAdult As Boolean
isAdult = True
```

## 4. 常量与变量

### 4.1 变量

变量是在程序运行过程中可以改变的量。

声明变量：

```vb
Dim 变量名 As 数据类型
```

示例：

```vb
Dim x As Integer
Dim y As Integer
Dim sum As Integer

x = 10
y = 20
sum = x + y
```

### 4.2 常量

常量是在程序运行过程中不改变的量。

声明常量：

```vb
Const PI As Double = 3.1415926
```

示例：

```vb
Private Sub Command1_Click()
    Const PI As Double = 3.1415926
    Dim r As Double
    Dim area As Double

    r = Val(Text1.Text)
    area = PI * r * r
    Label1.Caption = "圆面积为：" & area
End Sub
```

## 5. 变量作用域

作用域表示变量在哪些位置可以使用。

### 5.1 过程级变量

在过程内部声明，只能在该过程中使用：

```vb
Private Sub Command1_Click()
    Dim count As Integer
    count = 10
End Sub
```

### 5.2 窗体级变量

在窗体代码顶部声明，可以被该窗体中的多个过程使用：

```vb
Dim total As Integer

Private Sub Command1_Click()
    total = total + 1
End Sub

Private Sub Command2_Click()
    MsgBox total
End Sub
```

### 5.3 静态变量

`Static` 声明的变量在过程结束后仍保留原值：

```vb
Private Sub Command1_Click()
    Static count As Integer
    count = count + 1
    Label1.Caption = "点击次数：" & count
End Sub
```

第一次点击时 `count` 为 1，第二次点击时 `count` 为 2。

## 6. 运算符

### 6.1 算术运算符

| 运算符 | 含义 | 示例 | 结果 |
| --- | --- | --- | --- |
| `+` | 加 | `3 + 2` | 5 |
| `-` | 减 | `3 - 2` | 1 |
| `*` | 乘 | `3 * 2` | 6 |
| `/` | 实数除法 | `5 / 2` | 2.5 |
| `\` | 整数除法 | `5 \ 2` | 2 |
| `Mod` | 求余数 | `5 Mod 2` | 1 |
| `^` | 乘方 | `2 ^ 3` | 8 |

示例：

```vb
Dim a As Integer
Dim b As Integer

a = 17 \ 5
b = 17 Mod 5

MsgBox "商是：" & a & "，余数是：" & b
```

### 6.2 关系运算符

关系运算符用于比较，结果是 `True` 或 `False`。

| 运算符 | 含义 |
| --- | --- |
| `>` | 大于 |
| `<` | 小于 |
| `>=` | 大于等于 |
| `<=` | 小于等于 |
| `=` | 等于 |
| `<>` | 不等于 |

示例：

```vb
Dim score As Integer
score = 85

If score >= 60 Then
    MsgBox "及格"
End If
```

### 6.3 逻辑运算符

| 运算符 | 含义 | 说明 |
| --- | --- | --- |
| `And` | 与 | 两个条件都为真，结果才为真 |
| `Or` | 或 | 有一个条件为真，结果就为真 |
| `Not` | 非 | 取反 |

示例：

```vb
Dim score As Integer
score = 88

If score >= 80 And score <= 100 Then
    MsgBox "成绩优秀"
End If
```

### 6.4 字符串连接运算符

字符串连接推荐使用 `&`：

```vb
Dim name As String
Dim age As Integer

name = "小红"
age = 16

MsgBox name & "今年" & age & "岁"
```

## 7. 常用输入与输出

### 7.1 MsgBox 输出

`MsgBox` 用于弹出消息框。

```vb
MsgBox "欢迎使用本程序"
```

也可以显示变量：

```vb
Dim sum As Integer
sum = 100
MsgBox "总分为：" & sum
```

### 7.2 InputBox 输入

`InputBox` 用于弹出输入框，返回的是字符串。

```vb
Dim name As String
name = InputBox("请输入姓名：")
MsgBox "你好，" & name
```

如果要输入数字，需要转换：

```vb
Dim age As Integer
age = Val(InputBox("请输入年龄："))
```

### 7.3 文本框输入

窗体程序中更常用文本框输入：

```vb
Dim score As Single
score = Val(Text1.Text)
```

`Text1.Text` 是字符串，`Val` 可以把字符串转换为数值。

### 7.4 标签输出

```vb
Label1.Caption = "计算完成"
```

### 7.5 窗体 Print 输出

```vb
Private Sub Command1_Click()
    Print "欢迎学习 VB"
End Sub
```

`Print` 会在窗体上输出文字，但在图形界面程序中，标签输出更常见。

## 8. 常用类型转换函数

| 函数 | 作用 | 示例 |
| --- | --- | --- |
| `Val` | 把字符串转成数值 | `Val("123")` |
| `Str` | 把数值转成字符串 | `Str(123)` |
| `CInt` | 转成整数 | `CInt(3.6)` |
| `CLng` | 转成长整数 | `CLng(1000)` |
| `CSng` | 转成 Single | `CSng("3.14")` |
| `CDbl` | 转成 Double | `CDbl("3.1415")` |
| `CStr` | 转成字符串 | `CStr(99)` |
| `Format` | 按格式输出 | `Format(3.1415, "0.00")` |

示例：

```vb
Dim price As Double
price = 12.3456
Label1.Caption = Format(price, "0.00")
```

输出结果为：

```text
12.35
```

# 三、顺序结构程序设计

## 1. 什么是顺序结构

顺序结构是最简单的程序结构，程序按照语句出现的先后顺序依次执行。

例如：

```vb
a = 10
b = 20
c = a + b
MsgBox c
```

执行顺序是：

1. 把 10 赋值给 `a`。
2. 把 20 赋值给 `b`。
3. 计算 `a + b`，赋值给 `c`。
4. 显示 `c`。

## 2. 赋值语句

赋值语句的一般格式：

```vb
变量 = 表达式
```

示例：

```vb
x = 5
y = x + 3
name = "张三"
```

注意：赋值号右边先计算，计算结果再赋给左边变量。

```vb
x = 10
x = x + 1
```

第二句的意思不是数学方程，而是：

1. 取出 `x` 原来的值 10。
2. 计算 `10 + 1`。
3. 把结果 11 重新存入 `x`。

## 3. 顺序结构示例一：两个数求和

### 3.1 界面设计

| 控件 | Name | 说明 |
| --- | --- | --- |
| TextBox | Text1 | 输入第一个数 |
| TextBox | Text2 | 输入第二个数 |
| CommandButton | Command1 | 计算 |
| Label | Label1 | 显示结果 |

### 3.2 代码

```vb
Private Sub Command1_Click()
    Dim a As Double
    Dim b As Double
    Dim sum As Double

    a = Val(Text1.Text)
    b = Val(Text2.Text)
    sum = a + b

    Label1.Caption = "两数之和为：" & sum
End Sub
```

### 3.3 分析

程序从文本框获取两个数，依次执行输入、计算、输出三个步骤。

这是典型的顺序结构：

```text
输入数据 -> 处理数据 -> 输出结果
```

## 4. 顺序结构示例二：计算圆的面积和周长

### 4.1 问题分析

已知圆的半径 `r`，计算：

```text
面积 = π × r × r
周长 = 2 × π × r
```

### 4.2 代码

```vb
Private Sub Command1_Click()
    Const PI As Double = 3.1415926
    Dim r As Double
    Dim area As Double
    Dim perimeter As Double

    r = Val(Text1.Text)
    area = PI * r * r
    perimeter = 2 * PI * r

    Label1.Caption = "面积：" & Format(area, "0.00")
    Label2.Caption = "周长：" & Format(perimeter, "0.00")
End Sub
```

### 4.3 重点

1. 用 `Const` 定义不会改变的圆周率。
2. 用 `Val` 把文本框内容转换成数值。
3. 用 `Format` 控制小数位数。

## 5. 顺序结构示例三：摄氏温度转华氏温度

### 5.1 公式

```text
华氏温度 = 摄氏温度 × 9 / 5 + 32
```

### 5.2 代码

```vb
Private Sub Command1_Click()
    Dim c As Double
    Dim f As Double

    c = Val(Text1.Text)
    f = c * 9 / 5 + 32

    Label1.Caption = "华氏温度：" & Format(f, "0.0")
End Sub
```

## 6. 顺序结构示例四：交换两个变量的值

交换两个变量的值需要使用临时变量。

```vb
Private Sub Command1_Click()
    Dim a As Integer
    Dim b As Integer
    Dim temp As Integer

    a = Val(Text1.Text)
    b = Val(Text2.Text)

    temp = a
    a = b
    b = temp

    Label1.Caption = "交换后 a=" & a & "，b=" & b
End Sub
```

如果没有 `temp`，一个变量的原值可能会被覆盖。

## 7. 顺序结构常见问题

### 7.1 忘记类型转换

错误写法：

```vb
sum = Text1.Text + Text2.Text
```

当文本框内容是字符串时，可能出现字符串连接或类型错误。

推荐写法：

```vb
sum = Val(Text1.Text) + Val(Text2.Text)
```

### 7.2 使用变量前没有赋值

```vb
Dim total As Integer
Label1.Caption = total
```

在 VB 中，数值变量默认值通常是 0，但仍然建议先明确赋值再使用。

### 7.3 公式书写错误

数学中的：

```text
1
-
2
```

在 VB 中必须写成：

```vb
1 / 2
```

数学中的：

```text
ab
```

在 VB 中必须写成：

```vb
a * b
```

# 四、选择结构程序设计

## 1. 什么是选择结构

选择结构根据条件是否成立，决定执行哪一部分代码。

生活中的选择：

```text
如果下雨，就带伞；否则，不带伞。
```

程序中的选择：

```vb
If raining = True Then
    MsgBox "带伞"
Else
    MsgBox "不带伞"
End If
```

## 2. 单分支 If 语句

格式：

```vb
If 条件 Then
    语句组
End If
```

含义：如果条件成立，就执行语句组；如果条件不成立，就跳过。

示例：判断是否及格。

```vb
Private Sub Command1_Click()
    Dim score As Integer
    score = Val(Text1.Text)

    If score >= 60 Then
        Label1.Caption = "及格"
    End If
End Sub
```

如果输入 80，会显示“及格”。如果输入 50，标签不会被修改。

## 3. 双分支 If...Else 语句

格式：

```vb
If 条件 Then
    语句组1
Else
    语句组2
End If
```

含义：

- 条件成立，执行语句组 1。
- 条件不成立，执行语句组 2。

示例：

```vb
Private Sub Command1_Click()
    Dim score As Integer
    score = Val(Text1.Text)

    If score >= 60 Then
        Label1.Caption = "及格"
    Else
        Label1.Caption = "不及格"
    End If
End Sub
```

## 4. 多分支 If...ElseIf 语句

格式：

```vb
If 条件1 Then
    语句组1
ElseIf 条件2 Then
    语句组2
ElseIf 条件3 Then
    语句组3
Else
    语句组4
End If
```

示例：根据成绩判断等级。

```vb
Private Sub Command1_Click()
    Dim score As Integer
    score = Val(Text1.Text)

    If score >= 90 Then
        Label1.Caption = "优秀"
    ElseIf score >= 80 Then
        Label1.Caption = "良好"
    ElseIf score >= 70 Then
        Label1.Caption = "中等"
    ElseIf score >= 60 Then
        Label1.Caption = "及格"
    Else
        Label1.Caption = "不及格"
    End If
End Sub
```

### 4.1 为什么判断顺序很重要

下面写法是错误逻辑：

```vb
If score >= 60 Then
    Label1.Caption = "及格"
ElseIf score >= 80 Then
    Label1.Caption = "良好"
End If
```

如果 `score = 85`，第一个条件 `score >= 60` 已经成立，程序不会再判断 `score >= 80`，所以会错误显示“及格”。

正确写法应该从高分到低分判断。

## 5. 嵌套 If 语句

If 语句内部还可以再写 If。

示例：判断用户名和密码。

```vb
Private Sub Command1_Click()
    Dim userName As String
    Dim password As String

    userName = Text1.Text
    password = Text2.Text

    If userName = "admin" Then
        If password = "123456" Then
            Label1.Caption = "登录成功"
        Else
            Label1.Caption = "密码错误"
        End If
    Else
        Label1.Caption = "用户名不存在"
    End If
End Sub
```

嵌套 If 不宜过多，否则程序难读。能用逻辑运算符或 `Select Case` 简化时，可以适当简化。

## 6. Select Case 语句

格式：

```vb
Select Case 表达式
```

```vb
Private Sub Command1_Click()
```

```vb
n Mod 2
```

## 9. 选择结构示例二：求三个数中的最大值

```vb
Private Sub Command1_Click()
```

# 五、循环结构程序设计

## 1. 什么是循环结构

循环结构用于重复执行一段代码。

例如，输出 1 到 10，如果不用循环，需要写 10 行：

```vb
Print 1
Print 2
Print 3
Print 4
Print 5
Print 6
Print 7
Print 8
Print 9
Print 10
```

使用循环后：

```vb
Dim i As Integer

For i = 1 To 10
    Print i
Next i
```

循环可以减少重复代码，提高程序效率。

## 2. For...Next 循环

`For...Next` 适合循环次数已知的情况。

格式：

```vb
For 循环变量 = 初值 To 终值 Step 步长
    循环体
Next 循环变量
```

其中 `Step 步长` 可以省略，默认步长为 1。

示例：

```vb
Dim i As Integer

For i = 1 To 5
    Print i
Next i
```

输出：

```text
1
2
3
4
5
```

## 3. Step 步长

### 3.1 步长为 2

输出 1、3、5、7、9：

```vb
Dim i As Integer

For i = 1 To 9 Step 2
    Print i
Next i
```

### 3.2 递减循环

输出 10 到 1：

```vb
Dim i As Integer

For i = 10 To 1 Step -1
    Print i
Next i
```

如果起点大于终点，必须使用负步长，否则循环不会执行。

## 4. For 循环示例一：求 1 到 100 的和

```vb
Private Sub Command1_Click()
    Dim i As Integer
    Dim sum As Integer

    sum = 0

    For i = 1 To 100
        sum = sum + i
    Next i

    Label1.Caption = "1 到 100 的和为：" & sum
End Sub
```

### 4.1 程序分析

`sum = sum + i` 是累加语句。

循环过程：

| i | sum 原值 | sum + i | sum 新值 |
| --- | --- | --- | --- |
| 1 | 0 | 1 | 1 |
| 2 | 1 | 3 | 3 |
| 3 | 3 | 6 | 6 |
| ... | ... | ... | ... |

## 5. For 循环示例二：求 1 到 100 的偶数和

方法一：使用 Step。

```vb
Private Sub Command1_Click()
    Dim i As Integer
    Dim sum As Integer

    sum = 0

    For i = 2 To 100 Step 2
        sum = sum + i
    Next i

    Label1.Caption = "1 到 100 的偶数和为：" & sum
End Sub
```

方法二：使用 If 判断。

```vb
Private Sub Command1_Click()
    Dim i As Integer
    Dim sum As Integer

    sum = 0

    For i = 1 To 100
        If i Mod 2 = 0 Then
            sum = sum + i
        End If
    Next i

    Label1.Caption = "1 到 100 的偶数和为：" & sum
End Sub
```

## 6. For 循环示例三：阶乘

阶乘定义：

```text
n! = 1 × 2 × 3 × ... × n
```

代码：

```vb
Private Sub Command1_Click()
    Dim n As Integer
    Dim i As Integer
    Dim result As Long

    n = Val(Text1.Text)
    result = 1

    For i = 1 To n
        result = result * i
    Next i

    Label1.Caption = n & " 的阶乘为：" & result
End Sub
```

注意：阶乘增长很快，`Integer` 容易溢出，所以结果变量使用 `Long`。

## 7. Do While...Loop 循环

`Do While...Loop` 适合循环次数不确定，但循环条件明确的情况。

格式：

```vb
Do While 条件
    循环体
Loop
```

含义：当条件为真时反复执行循环体。

示例：输出 1 到 5。

```vb
Dim i As Integer

i = 1

Do While i <= 5
    Print i
    i = i + 1
Loop
```

## 8. Do Until...Loop 循环

格式：

```vb
Do Until 条件
    循环体
Loop
```

含义：直到条件成立才停止。也可以理解为：当条件不成立时继续循环。

示例：

```vb
Dim i As Integer

i = 1

Do Until i > 5
    Print i
    i = i + 1
Loop
```

这段代码也会输出 1 到 5。

## 9. While...Wend 循环

VB 中也有 `While...Wend`：

```vb
Dim i As Integer

i = 1

While i <= 5
    Print i
    i = i + 1
Wend
```

不过在实际学习中，更推荐使用 `Do While...Loop`，结构更清晰。

## 10. 循环控制变量

循环结构通常包含三个关键部分：

1. 初始化：给循环变量初值。
2. 条件判断：决定是否继续循环。
3. 更新变量：让循环逐步接近结束。

示例：

```vb
i = 1              '初始化
Do While i <= 10   '条件判断
    Print i
    i = i + 1      '更新变量
Loop
```

如果忘记更新变量，可能造成死循环：

```vb
i = 1
Do While i <= 10
    Print i
Loop
```

这段代码中 `i` 永远是 1，条件永远成立，程序会一直循环。

## 11. Exit For 与 Exit Do

### 11.1 Exit For

提前退出 `For` 循环。

```vb
Dim i As Integer

For i = 1 To 10
    If i = 6 Then
        Exit For
    End If
    Print i
Next i
```

输出：

```text
1
2
3
4
5
```

### 11.2 Exit Do

提前退出 `Do` 循环。

```vb
Dim n As Integer

Do
    n = Val(InputBox("请输入一个正数，输入 0 结束："))
    If n = 0 Then
        Exit Do
    End If
    Print n
Loop
```

## 12. 嵌套循环

循环内部还可以再写循环。

示例：输出乘法口诀表。

```vb
Private Sub Command1_Click()
    Dim i As Integer
    Dim j As Integer

    For i = 1 To 9
        For j = 1 To i
            Print j & "×" & i & "=" & i * j;
        Next j
        Print
    Next i
End Sub
```

说明：

- 外层循环控制行数。
- 内层循环控制每一行的列数。
- `Print ...;` 中的分号表示继续在当前行输出。
- 单独的 `Print` 表示换行。

如果你的 VB 环境不方便显示乘号，也可以写成：

```vb
Print j & "*" & i & "=" & i * j;
```

## 13. 循环示例：判断素数

素数是只能被 1 和它本身整除的大于 1 的整数。

```vb
Private Sub Command1_Click()
    Dim n As Integer
    Dim i As Integer
    Dim isPrime As Boolean

    n = Val(Text1.Text)
    isPrime = True

    If n <= 1 Then
        isPrime = False
    Else
        For i = 2 To n - 1
            If n Mod i = 0 Then
                isPrime = False
                Exit For
            End If
        Next i
    End If

    If isPrime Then
        Label1.Caption = n & " 是素数"
    Else
        Label1.Caption = n & " 不是素数"
    End If
End Sub
```

优化写法：只需要判断到平方根附近。

```vb
Private Sub Command1_Click()
    Dim n As Integer
    Dim i As Integer
    Dim isPrime As Boolean

    n = Val(Text1.Text)
    isPrime = True

    If n <= 1 Then
        isPrime = False
    Else
        For i = 2 To Sqr(n)
            If n Mod i = 0 Then
                isPrime = False
                Exit For
            End If
        Next i
    End If

    If isPrime Then
        Label1.Caption = n & " 是素数"
    Else
        Label1.Caption = n & " 不是素数"
    End If
End Sub
```

# 六、数组

## 1. 什么是数组

数组是一组相同类型数据的集合，它们使用同一个数组名，通过下标区分不同元素。

如果要保存 5 个学生成绩，不使用数组可能要写：

```vb
Dim s1 As Integer
Dim s2 As Integer
Dim s3 As Integer
Dim s4 As Integer
Dim s5 As Integer
```

使用数组：

```vb
Dim score(1 To 5) As Integer
```

这样就可以使用：

```vb
score(1)
score(2)
score(3)
score(4)
score(5)
```

## 2. 数组的声明

### 2.1 固定大小数组

格式：

```vb
Dim 数组名(下界 To 上界) As 数据类型
```

示例：

```vb
Dim score(1 To 5) As Integer
```

表示声明了 5 个整数元素：

```text
score(1), score(2), score(3), score(4), score(5)
```

也可以省略下界：

```vb
Dim score(5) As Integer
```

在很多 VB 环境中，这表示下标从 0 到 5，共 6 个元素。为了避免混乱，初学阶段推荐明确写出 `1 To 5`。

### 2.2 字符串数组

```vb
Dim names(1 To 3) As String

names(1) = "张三"
names(2) = "李四"
names(3) = "王五"
```

### 2.3 小数数组

```vb
Dim price(1 To 10) As Double
```

## 3. 数组元素的访问

数组元素可以像普通变量一样使用。

```vb
Dim a(1 To 3) As Integer

a(1) = 10
a(2) = 20
a(3) = a(1) + a(2)

MsgBox a(3)
```

输出：

```text
30
```

## 4. 使用循环处理数组

数组和循环经常一起使用。

示例：输入 5 个成绩并求总分。

```vb
Private Sub Command1_Click()
    Dim score(1 To 5) As Integer
    Dim i As Integer
    Dim total As Integer

    total = 0

    For i = 1 To 5
        score(i) = Val(InputBox("请输入第 " & i & " 个学生成绩："))
        total = total + score(i)
    Next i

    Label1.Caption = "总分为：" & total
End Sub
```

## 5. 数组示例一：求平均分

```vb
Private Sub Command1_Click()
    Dim score(1 To 5) As Integer
    Dim i As Integer
    Dim total As Integer
    Dim average As Double

    total = 0

    For i = 1 To 5
        score(i) = Val(InputBox("请输入第 " & i & " 个成绩："))
        total = total + score(i)
    Next i

    average = total / 5
    Label1.Caption = "平均分为：" & Format(average, "0.00")
End Sub
```

## 6. 数组示例二：求最大值和最小值

```vb
Private Sub Command1_Click()
    Dim score(1 To 5) As Integer
    Dim i As Integer
    Dim maxScore As Integer
    Dim minScore As Integer

    For i = 1 To 5
        score(i) = Val(InputBox("请输入第 " & i & " 个成绩："))
    Next i

    maxScore = score(1)
    minScore = score(1)

    For i = 2 To 5
        If score(i) > maxScore Then
            maxScore = score(i)
        End If

        If score(i) < minScore Then
            minScore = score(i)
        End If
    Next i

    Label1.Caption = "最高分：" & maxScore & "，最低分：" & minScore
End Sub
```

程序思想：

1. 先假设第一个元素既是最大值，也是最小值。
2. 从第二个元素开始依次比较。
3. 遇到更大的数就更新最大值。
4. 遇到更小的数就更新最小值。

## 7. 数组示例三：统计及格人数

```vb
Private Sub Command1_Click()
```

## 8. 数组示例四：查找指定数据

目标：在数组中查找是否存在某个成绩。

```vb
Private Sub Command1_Click()
```

这里使用 `found` 作为标志变量。标志变量常用于记录某件事情是否发生。

## 9. 数组示例五：冒泡排序

冒泡排序是一种简单排序方法。它会多次比较相邻两个元素，如果顺序不对就交换。

目标：把 5 个数按从小到大排序。

```vb
Private Sub Command1_Click()
    Dim a(1 To 5) As Integer
    Dim i As Integer
    Dim j As Integer
    Dim temp As Integer
    Dim result As String

    For i = 1 To 5
        a(i) = Val(InputBox("请输入第 " & i & " 个数："))
    Next i

    For i = 1 To 4
        For j = 1 To 5 - i
            If a(j) > a(j + 1) Then
                temp = a(j)
                a(j) = a(j + 1)
                a(j + 1) = temp
            End If
        Next j
    Next i

    result = ""

    For i = 1 To 5
        result = result & a(i) & " "
    Next i

    Label1.Caption = "排序结果：" & result
End Sub
```

### 9.1 冒泡排序过程理解

假设数组为：

```text
5 2 4 1 3
```

第一轮比较：

1. 比较 5 和 2，交换：2 5 4 1 3
2. 比较 5 和 4，交换：2 4 5 1 3
3. 比较 5 和 1，交换：2 4 1 5 3
4. 比较 5 和 3，交换：2 4 1 3 5

第一轮结束后，最大数 5 到了最后。

继续进行第二轮、第三轮，最终得到：

```text
1 2 3 4 5
```

## 10. 动态数组

如果数组大小在程序运行时才知道，可以使用动态数组。

声明动态数组：

```vb
Dim score() As Integer
```

运行时重新定义大小：

```vb
ReDim score(1 To n)
```

示例：

```vb
Private Sub Command1_Click()
    Dim score() As Integer
    Dim n As Integer
    Dim i As Integer
    Dim total As Integer

    n = Val(InputBox("请输入学生人数："))
    ReDim score(1 To n)

    total = 0

    For i = 1 To n
        score(i) = Val(InputBox("请输入第 " & i & " 个成绩："))
        total = total + score(i)
    Next i

    Label1.Caption = "总分为：" & total
End Sub
```

## 11. ReDim Preserve

普通 `ReDim` 会重新分配数组空间，原来的数据会丢失。

如果想改变数组大小并保留原有数据，可以使用：

```vb
ReDim Preserve 数组名(新下界 To 新上界)
```

示例：

```vb
Dim a() As Integer
Dim i As Integer

ReDim a(1 To 3)
a(1) = 10
a(2) = 20
a(3) = 30

ReDim Preserve a(1 To 5)
a(4) = 40
a(5) = 50

For i = 1 To 5
    Print a(i)
Next i
```

注意：在经典 VB 中，多维数组使用 `ReDim Preserve` 时，通常只能改变最后一维的上界。

## 12. 二维数组

二维数组可以理解为表格，有行和列。

声明格式：

```vb
Dim 数组名(行下界 To 行上界, 列下界 To 列上界) As 数据类型
```

示例：保存 3 个学生的 2 门成绩。

```vb
Dim score(1 To 3, 1 To 2) As Integer
```

含义：

| 元素 | 含义 |
| --- | --- |
| `score(1, 1)` | 第 1 个学生第 1 门成绩 |
| `score(1, 2)` | 第 1 个学生第 2 门成绩 |
| `score(2, 1)` | 第 2 个学生第 1 门成绩 |
| `score(2, 2)` | 第 2 个学生第 2 门成绩 |
| `score(3, 1)` | 第 3 个学生第 1 门成绩 |
| `score(3, 2)` | 第 3 个学生第 2 门成绩 |

示例：输入并输出二维数组。

```vb
Private Sub Command1_Click()
    Dim score(1 To 3, 1 To 2) As Integer
    Dim i As Integer
    Dim j As Integer
    Dim result As String

    For i = 1 To 3
        For j = 1 To 2
            score(i, j) = Val(InputBox("请输入第 " & i & " 个学生第 " & j & " 门成绩："))
        Next j
    Next i

    result = ""

    For i = 1 To 3
        result = result & "第 " & i & " 个学生："
        For j = 1 To 2
            result = result & score(i, j) & " "
        Next j
        result = result & vbCrLf
    Next i

    MsgBox result
End Sub
```

## 练习二：求 n 以内所有奇数的和

输入一个正整数 `n`，求 1 到 `n` 之间所有奇数的和。

### 参考代码

```vb
Private Sub Command1_Click()
    Dim n As Integer
    Dim i As Integer
    Dim sum As Integer

    n = Val(Text1.Text)
    sum = 0

    For i = 1 To n Step 2
        sum = sum + i
    Next i

    Label1.Caption = "奇数和为：" & sum
End Sub
```

## 练习三：统计一组成绩

### 要求

输入若干名学生成绩，统计：

1. 总分。
2. 平均分。
3. 最高分。
4. 最低分。
5. 及格人数。

### 参考代码

```vb
Private Sub Command1_Click()
    Dim score() As Integer
    Dim n As Integer
    Dim i As Integer
    Dim total As Integer
    Dim average As Double
    Dim maxScore As Integer
    Dim minScore As Integer
    Dim passCount As Integer

    n = Val(InputBox("请输入学生人数："))
    ReDim score(1 To n)

    total = 0
    passCount = 0

    For i = 1 To n
        score(i) = Val(InputBox("请输入第 " & i & " 个成绩："))
        total = total + score(i)

        If score(i) >= 60 Then
            passCount = passCount + 1
        End If
    Next i

    maxScore = score(1)
    minScore = score(1)

    For i = 2 To n
        If score(i) > maxScore Then
            maxScore = score(i)
        End If

        If score(i) < minScore Then
            minScore = score(i)
        End If
    Next i

    average = total / n

    MsgBox "总分：" & total & vbCrLf & _
           "平均分：" & Format(average, "0.00") & vbCrLf & _
           "最高分：" & maxScore & vbCrLf & _
           "最低分：" & minScore & vbCrLf & _
           "及格人数：" & passCount
End Sub
```

## 练习四：猜数字游戏

### 要求

程序随机生成 1 到 100 之间的整数，用户不断输入猜测值，程序提示“猜大了”“猜小了”或“猜对了”。

### 参考代码

```vb
Private Sub Command1_Click()
    Dim answer As Integer
    Dim guess As Integer
    Dim count As Integer

    Randomize
    answer = Int(Rnd * 100) + 1
    count = 0

    Do
        guess = Val(InputBox("请输入 1 到 100 之间的整数："))
        count = count + 1

        If guess > answer Then
            MsgBox "猜大了"
        ElseIf guess < answer Then
            MsgBox "猜小了"
        Else
            MsgBox "猜对了，共猜了 " & count & " 次"
            Exit Do
        End If
    Loop
End Sub
```

# 八、常见错误与调试方法

## 1. 常见错误

### 1.1 变量名拼写不一致

```vb
Dim totalScore As Integer
totalscore = 100
```

VB 对大小写通常不敏感，但如果拼写不同，就可能导致错误。建议开启变量声明要求。

### 1.2 忘记 End If 或 Next

错误示例：

```vb
If score >= 60 Then
    Label1.Caption = "及格"
```

正确示例：

```vb
If score >= 60 Then
    Label1.Caption = "及格"
End If
```

循环也必须有 `Next`：

```vb
For i = 1 To 10
    Print i
Next i
```

### 1.3 条件判断写错

错误示例：

```vb
If 60 <= score <= 100 Then
    Label1.Caption = "合法成绩"
End If
```

VB 中不能这样连续比较。正确写法：

```vb
If score >= 60 And score <= 100 Then
    Label1.Caption = "合法成绩"
End If
```

### 1.4 数组下标越界

```vb
Dim a(1 To 5) As Integer
a(6) = 100
```

数组只有 `a(1)` 到 `a(5)`，访问 `a(6)` 会越界。

### 1.5 死循环

```vb
i = 1
Do While i <= 10
    Print i
Loop
```

缺少：

```vb
i = i + 1
```

### 1.6 除数为 0

```vb
result = a / b
```

如果 `b = 0`，程序会出错。应先判断：

```vb
If b = 0 Then
    Label1.Caption = "除数不能为 0"
Else
    result = a / b
End If
```

## 2. 调试方法

### 2.1 使用 MsgBox 查看变量值

```vb
MsgBox "当前 i 的值是：" & i
```

适合初学者快速检查程序执行过程。

### 2.2 使用 Print 输出中间结果

```vb
Print "sum=" & sum
```

### 2.3 分段检查

复杂程序不要一次写完后再运行。推荐：

1. 先完成输入。
2. 再完成基本计算。
3. 再增加选择或循环。
4. 最后处理边界情况。

### 2.4 检查边界数据

测试程序时，不只测试普通数据，还要测试边界数据。

例如成绩等级程序应测试：

```text
-1, 0, 59, 60, 69, 70, 79, 80, 89, 90, 100, 101
```

这些数据可以帮助发现条件判断是否正确。

## 3. 学习建议

1. 先理解三大结构：顺序、选择、循环。
2. 每个示例都手动输入一遍，不要只复制。
3. 改动示例中的数字、条件和输出内容，观察程序变化。
4. 学数组时一定要配合循环，否则数组的优势体现不出来。
5. 写程序前先列出“输入、处理、输出”。
6. 程序出错时先读错误提示，再检查变量、条件和循环边界。
